# AI Travel Planner

An AI-powered travel planning application built with Next.js, Supabase, and Google Gemini API.

## Features
- User Authentication
- Create and Manage Trips
- AI Generated Travel Itineraries
- Supabase Database Integration
- Responsive UI

## Tech Stack
- Next.js
- TypeScript
- Supabase
- Google Gemini API
- Tailwind CSS

## Setup

1. Clone repository
2. Install dependencies
   npm install
3. Create .env.local
4. Add Supabase and Gemini API keys
5. Run
   npm run dev
